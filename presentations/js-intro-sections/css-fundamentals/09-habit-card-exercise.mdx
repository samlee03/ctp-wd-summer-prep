describe('Habit Card Styling', () => {
    const card = document.querySelector('.habit-card');
    const styles = window.getComputedStyle(card);
    
    it('should have rounded corners', () => {
        expect(styles.borderRadius).to.not.equal('0px');
    });
    
    it('should have a shadow', () => {
        expect(styles.boxShadow).to.not.equal('none');
    });
    
    it('should use flexbox or grid', () => {
        const display = styles.display;
        expect(['flex', 'grid'].includes(display)).to.be.true;
    });
    
    it('should style the complete button with primary color', () => {
        const btn = document.querySelector('.btn-complete');
        const btnStyles = window.getComputedStyle(btn);
        expect(btnStyles.backgroundColor).to.include('76, 175, 80'); // RGB for #4CAF50
    });
});

<Notes>
This comprehensive exercise brings together everything students have learned about CSS. They start with semantic HTML structure and must create a polished, professional habit card component. This mirrors real-world development where you receive a design and must implement it with clean, maintainable CSS.

The exercise teaches component thinking - creating reusable styles that could work for any habit card, not just this specific example. Students must consider the visual hierarchy: the title should draw attention, the category provides context, stats show progress, and the action button should be prominent.

Key challenges include: using Flexbox or Grid for the layout, creating a cohesive color scheme, adding subtle shadows for depth, ensuring proper spacing throughout, making it responsive for different screen sizes, and adding hover states for interactivity. Students should also consider accessibility: sufficient color contrast, focus states for keyboard navigation, and semantic HTML.

This exercise works best when students share their solutions and discuss different approaches. There's no single "correct" solution - different designs can all be effective. The goal is to create something that looks professional, works well, and demonstrates understanding of CSS principles.
